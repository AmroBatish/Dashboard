# 📊 Responsive Admin Dashboard

A responsive admin dashboard built from scratch using **HTML and CSS** as part of my front-end learning journey.

---

## 📖 About The Project

This project is a **practice admin dashboard** that I built while learning front-end development and following tutorials by Elzero Web School.

The main goal of this project was to practice building a **complete dashboard interface** and improve my skills in layout design, responsive development, and UI structuring **without using any external libraries or frameworks**.

The dashboard simulates a real admin panel and includes multiple components commonly used in modern dashboards.

---

## 🚀 Features

- 📱 **Responsive Design**  
  The layout adapts smoothly to desktop, tablet, and mobile screens.

- 🎨 **Clean UI Layout**  
  A simple and modern dashboard interface focused on readability and usability.

- 🧩 **Multiple Dashboard Widgets**
  - Welcome card
  - Quick draft section
  - Yearly targets progress bars
  - Ticket statistics
  - Latest news
  - Latest tasks

- 🧱 **Built From Scratch**  
  The entire project was built using **pure HTML and CSS (Vanilla CSS)** without frameworks like Bootstrap or Tailwind.

- ⚙️ **Custom CSS Structure**  
  I wrote all the styles manually and experimented with creating a **lightweight CSS structure** to keep the code organized and scalable.

---

## 🎯 Learning Goals

Through this project I practiced:

- Building **dashboard layouts**
- Using **Flexbox and CSS Grid**
- Creating **responsive designs**
- Structuring CSS for better **maintainability**
- Designing reusable **UI components**
